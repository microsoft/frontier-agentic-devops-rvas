# Azure SRE Agent Runbooks

These assets support the Azure SRE Agent controlled-incident fallback path.

- `challenge-04-incident-packet.md` gives participants a simulated Grubify production signal.
- `challenge-04-triage-template.md` captures the operational response.
- `generated/` may be created by local simulation scripts and is intentionally ignored by Git.

Use live Azure SRE Agent evidence when available. When it is not available, keep the same sequence: Azure signal, telemetry evidence, agent transcript, source-code lead, remediation work with evidence and human approval, recovery proof, and follow-up improvement.
